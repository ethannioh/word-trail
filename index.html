使用說明

適合放在 GitHub 的靜態網站

1. 先新增自己的英文單字、中文意思與例句。

2. 到期單字會出現在「今日複習」。

3. 每次評分後，系統會重新安排下次複習時間。

4. 將本專案放到 GitHub Pages、Netlify 或其他靜態空間即可使用。
